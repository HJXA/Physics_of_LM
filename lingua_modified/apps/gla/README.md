# Physics of Language Models: Part 4.2, Canon Layers at Scale where Synthetic Pretraining Resonates in Reality

**Author**: Zeyuan Allen-Zhu  

This folder introduces new code, serving as the entry point for pretraining **HuggingFace-compatible** models using the **Lingua** codebase.  
Specifically, this entry point has been used to pretrain our [modified implementations](../../huggingface/) of **GLA**, **GDN**, and **Mamba2** models, all enhanced with **Canon Layers** and additional architectural improvements.  

For detailed usage instructions, please refer to the [root folder](../../) of the Lingua codebase.
